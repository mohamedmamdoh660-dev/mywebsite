# توثيق موقع قناة Book Breakdown AR

موقع تعريفي وتفاعلي كامل باللغة العربية لقناة اليوتيوب **Book Breakdown AR** (@BookBreakdownAR). تم بناء الموقع بالكامل كصفحة واحدة (Single Page Application) متجاوبة وتدعم التوجيه المبني على الهاش (Hash-based Routing).

## 2. نظام التصميم والتنسيقات (CSS Design System)

تم بناء التصميم باستخدام **Vanilla CSS** مع دعم كامل للاتجاه من اليمين إلى اليسار (RTL) وتفعيل الوضع المظلم كافتراضي مع إمكانية التحويل للوضع المضيء.

### الألوان الأساسية المستخدمة:
- **الوضع المظلم (Dark Mode - Default):**
  - الخلفية العامة: أسود فحم (`#0c0b0a`)
  - خلفية البطاقات والعناصر: بني فحم دافئ (`#141211`)
  - النص الأساسي: أوف وايت دافئ (`#f5f2eb`)
  - النص الفرعي / الباهت: بيج رمادي (`#b5af9f`)
  - اللون المميز (Accent Color): نحاسي/ذهبي دافئ (`#c8843a`)
  - لون الحدود: بني غامق جداً (`#24201d`)
  
- **الوضع المضيء (Light Mode):**
  - الخلفية العامة: كريمي فاتح دافئ (`#f7f5f0`)
  - خلفية البطاقات والعناصر: أبيض ناصع (`#ffffff`)
  - النص الأساسي: فحمي داكن (`#1c1a17`)
  - النص الفرعي / الباهت: بيج داكن (`#666052`)
  - اللون المميز (Accent Color): نحاسي/ذهبي دافئ (`#c8843a`)
  - لون الحدود: بيج كريمي دافئ (`#e6e0d3`)

### الخطوط المستخدمة:
- خط العناوين والشعارات: `Cairo` (من Google Fonts) لإعطاء طابع حاد وعريض متناسب مع الهوية الفكرية الصادمة للقناة.
- خط النصوص والجسم العام للموقع: `Tajawal` (من Google Fonts) لتسهيل القراءة وتوفير مظهر مريح وحديث للمقالات والفقرات.

## 3. الميزات التفاعلية (JS Logic)

يحتوي الموقع على العديد من الميزات التفاعلية المكتوبة بـ JavaScript الصافي:

### أ. نظام التوجيه بالهاش (Hash-based Routing)
تتنقل الصفحة بين 5 أقسام رئيسية بناءً على رابط الهاش المستدعى دون الحاجة لإعادة تحميل الصفحة أو استخدام خادم خلفي:
1. `#home` (الرئيسية)
2. `#episodes` (الحلقات الكاملة وأرشيف البحث وتصفية التصنيفات)
3. `#books` (مكتبة تشريح الكتب المصممة بنظام ثلاثي الأبعاد تفاعلي)
4. `#start` (دليل البدء التفاعلي لتصنيف واقتراح الحلقات حسب المشكلة الحالية للمشاهد)
5. `#about` (عن القناة ورسالتها)

### ب. شريط البحث والتصفية (Search & Filter)
في صفحة الحلقات (`#episodes`)، يتفاعل شريط البحث وتصنيف الأزرار (المال، ريادة الأعمال، علم النفس، الوعي الذاتي) بشكل فوري مع أرشيف الحلقات ليعرض النتائج المطابقة مع إشعار المستخدم في حال عدم وجود نتائج.

### ج. دليل التحرر المخصص (Start Here Quiz)
في صفحة ابدأ هنا (`#start`)، يختار المستخدم المشكلة التي يعاني منها (مثال: "أنا تعبان ماليًا") ليقوم المحرك بعرض قائمة مرتبة بخطوات تسلسلية (Step-by-Step Pathway) للحلقات التي تساهم في حل هذه المشكلة بالتحديد مع روابط مشاهدة سريعة.

### د. تجنب الفلاش اللوني وتفعيل الثيم (FOUC avoidance)
تم دمج كود برمجي صغير في وسم `<head>` يقرأ خيار المستخدم المخزن في الـ `localStorage` أو إعدادات النظام مباشرة ويطبقه قبل رندرة الصفحة لمنع حدوث وميض الضوء الأبيض عند تحميل الصفحة لمن يفضلون الوضع المظلم.

### هـ. تأثيرات التمرير (Scroll Reveal Animations)
تم استخدام ميزة `Scroll-driven animations` الحديثة في المتصفحات الداعمة كترقية تدريجية (Progressive Enhancement) وتوفير بديل عالي الأداء باستخدام `IntersectionObserver` للمتصفحات القديمة أو غير الداعمة كـ Firefox.

## 4. كيفية تحديث البيانات والمحتوى

تم تجميع بيانات المحتوى في بداية وسم الـ `<script>` المدمج في نهاية ملف `book-breakdown-ar.html` لتسهيل الصيانة والتحديث:

- لتحديث الحلقات، قم بالتعديل على مصفوفة الكائنات `EPISODES_DATA` وإضافة حلقة جديدة بالشكل التالي:
```javascript
{
  id: "e7",
  title: "عنوان الفيديو الجديد هنا",
  category: "المال", // أو ريادة الأعمال، علم النفس، الوعي الذاتي
  youtubeId: "YOUTUBE_VIDEO_ID",
  duration: "15:30",
  views: "1.5K",
  description: "وصف موجز للمحتوى الجريء والأنظمة المفككة في هذه الحلقة."
}
```

- لتحديث الكتب المغطاة، قم بالتعديل على مصفوفة الكائنات `BOOKS_DATA`:
```javascript
{
  title: "اسم الكتاب بالكامل",
  author: "اسم الكاتب بالكامل",
  category: "التصنيف",
  episodeLinkId: "e7", // معرف الحلقة المرتبطة به في مصفوفة الحلقات
  coverBg: "#172722", // لون خلفية تصميم غلاف الكتاب المخصص
  coverText: "عنوان مختصر يظهر على الغلاف المطبوع"
}
```
